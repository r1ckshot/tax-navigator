# Tax Navigator

Податковий навігатор для українців у Польщі з крос-бордерним доходом (укр ФОП, B2B-оффер, віддалена укр-зарплата).

## Проблема
Українець у Польщі не може без консультації за 200–300 zł відповісти на три питання: чи я податковий резидент Польщі, яка форма мені вигідна і легальна (укр ФОП / JDG / інкубатор / UoP), що і куди подавати в обох країнах. Безкоштовна інформація існує, але фрагментована й ніколи не персоналізована.

## Продукт
Веб-застосунок (Next.js), mobile-first, українською.
1. **Безкоштовно:** анкета-навігатор (~10 питань) → вердикт резидентства + порівняння сценаріїв з діапазонами "на руки".
2. **Платно:** персональний план дій (PDF) — кроки, документи, дедлайни обох країн, KSeF-календар.
3. **v2:** підписка на моніторинг змін правил.

Продукт не подає декларації за користувача і не дає персональних рекомендацій — тільки діапазони, порівняння варіантів і цитати джерел.
